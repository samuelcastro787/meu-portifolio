# 💼 Portfólio - Samuel Castro

Portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e informações de contato como **Desenvolvedor Back-End em formação**.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5**
* **CSS3**
* **JavaScript**
* **Font Awesome**
* **Google Fonts**

---

## 🎯 Funcionalidades

O portfólio possui diversas funcionalidades modernas:

* Navegação com **scroll suave**
* **Menu responsivo** para dispositivos móveis
* **Animações ao rolar a página**
* Destaque automático da seção ativa no menu
* Mudança dinâmica de estilo do **header conforme a seção**
* Layout **totalmente responsivo**

---

## 📱 Responsividade

O site foi projetado para funcionar corretamente em diferentes tamanhos de tela:

* 💻 Desktop
* 📱 Tablets
* 📱 Smartphones

Para melhorar a experiência em telas menores:

* A imagem da seção principal é ocultada
* O menu se transforma em **menu hamburguer**
* Os grids se adaptam automaticamente
